# MyBatis 적용

<br>

### ItemMapper 인터페이스

```java
package hello.itemservice.repository.mybatis;

import hello.itemservice.domain.Item;
import hello.itemservice.repository.ItemSearchCond;
import hello.itemservice.repository.ItemUpdateDto;
import org.apache.ibatis.annotations.Mapper;
import org.apache.ibatis.annotations.Param;
import java.util.List;
import java.util.Optional;

@Mapper
public interface ItemMapper {
    void save(Item item);
    
    void update(@Param("id") Long id, @Param("updateParam") ItemUpdateDto updateParam);
    
    Optional<Item> findById(Long id);
    
    List<Item> findAll(ItemSearchCond itemSearch);
}
```

|||
| --- | --- |
| `@Mapper` | MyBatis 인식 애노테이션 |
| 메서드명 | XML의 `id` 속성과 매칭 |
| `@Param` | 파라미터 2개 이상일 때 이름 지정 필수 |
| 반환 타입 | XML의 `resultType` 과 일치해야 함 |

> **중요:** `@Mapper`가 있어야 resource의 같은 위치 xml파일을 보고 MyBatis가 인식함

<br>

### ItemMapper.xml - INSERT

```xml
<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE mapper PUBLIC "-//mybatis.org//DTD Mapper 3.0//EN"
    "http://mybatis.org/dtd/mybatis-3-mapper.dtd">

<mapper namespace="hello.itemservice.repository.mybatis.ItemMapper">
    
    <insert id="save" useGeneratedKeys="true" keyProperty="id">
        insert into item (item_name, price, quantity)
        values (#{itemName}, #{price}, #{quantity})
    </insert>
    
</mapper>
```

||||
| --- | --- | --- |
| `id` | "save" | ItemMapper의 메서드명과 일치 |
| `useGeneratedKeys` | "true" | DB가 생성하는 PK 사용 |
| `keyProperty` | "id" | PK를 매핑할 객체 필드명 |
| `#{itemName}` | 파라미터 | 객체의 필드명 사용 (getter 자동 호출) |

1. Item 객체 파라미터 전달
2. `#{itemName}` → `item.getItemName()` 자동 호출
3. INSERT 실행
4. 생성된 ID를 `item.id`에 자동 설정

<br>

### ItemMapper.xml - UPDATE

```xml
<update id="update">
    update item
    set item_name=#{updateParam.itemName},
        price=#{updateParam.price},
        quantity=#{updateParam.quantity}
    where id = #{id}
</update>
```

|||
| --- | --- |
| `#{updateParam.itemName}` | 중첩 객체 필드 접근 (도트 표기법) |
| `#{id}` | @Param("id")로 전달된 파라미터 |
| 매퍼 인터페이스 | `update(@Param("id") Long id, @Param("updateParam") ItemUpdateDto updateParam)` |

<br>

- 파라미터가 2개 이상이면 **반드시** `@Param` 필수임
- XML에서는 `@Param`의 이름으로 접근하니 주의해야함

<br>

### ItemMapper.xml - SELECT (단건)

```xml
<select id="findById" resultType="Item">
    select id, item_name, price, quantity
    from item
    where id = #{id}
</select>
```

|||
| --- | --- |
| `resultType="Item"` | 반환 타입 (패키지명 생략 가능) |
| snake_case 자동 변환 | `item_name` → `itemName` 자동 매핑 |
| BeanPropertyRowMapper | MyBatis가 내부적으로 자동 적용 |

<br>

### ItemMapper.xml - SELECT (동적 쿼리)

```xml
<select id="findAll" resultType="Item">
    select id, item_name, price, quantity
    from item
    <where>
        <if test="itemName != null and itemName != ''">
            and item_name like concat('%',#{itemName},'%')
        </if>
        <if test="maxPrice != null">
            and price &lt;= #{maxPrice}
        </if>
    </where>
</select>
```

|||
| --- | --- |
| `<where>` | WHERE 문 자동 생성, 첫 AND 제거 |
| `<if test="조건">` | 조건이 true면 해당 구문 포함 |

<br>

**케이스별 예시:**

**조건 없음**
```xml
<where>
<!-- 모든 <if> 실패 -->
</where>
<!-- <where> 제거 됨 -->
결과: select id, item_name, price, quantity from item
```

<br>

**itemName만 지정**
```xml
<where>
    <if test="itemName != null and itemName != ''">  <!-- 성공 -->
        and item_name like concat('%',#{itemName},'%')
    </if>
    <if test="maxPrice != null">  <!-- 실패 -->
        ...
    </if>
</where>
<!-- <where>가 첫 AND를 WHERE로 변환 -->
결과: select ... from item where item_name like concat('%','value','%')
```

<br>

**maxPrice만 지정**
```xml
<where>
    <if test="itemName != null and itemName != ''">  <!-- 실패 -->
        and item_name like concat('%',#{itemName},'%')
    </if>
    <if test="maxPrice != null">  <!-- 성공 -->
        and price &lt;= #{maxPrice}  <!-- &lt; = < -->
    </if>
</where>
결과: select ... from item where and price <= ...
```

<br>

**둘 다 지정**
```xml
<where>
    <if test="itemName != null and itemName != ''">  <!-- 성공 -->
        and item_name like concat('%',#{itemName},'%')
    </if>
    <if test="maxPrice != null">  <!-- 성공 -->
        and price &lt;= #{maxPrice}  <!-- &lt; = < -->
    </if>
</where>
결과: select ... from item where item_name like ... and price <= ...
```

<br>

**XML 특수 문자 처리:**
| 문자 | 엔티티 |
| --- | --- |
| `<` | `&lt;` |
| `>` | `&gt;` |
| `&` | `&amp;` |
| `'` | `&apos;` |
| `"` | `&quot;` |

## MyBatis - 설정과 실행

### MyBatisItemRepository

```java
package hello.itemservice.repository.mybatis;

import hello.itemservice.domain.Item;
import hello.itemservice.repository.ItemRepository;
import hello.itemservice.repository.ItemSearchCond;
import hello.itemservice.repository.ItemUpdateDto;
import lombok.RequiredArgsConstructor;
import org.springframework.stereotype.Repository;
import java.util.List;
import java.util.Optional;

@Repository
@RequiredArgsConstructor
public class MyBatisItemRepository implements ItemRepository {
    private final ItemMapper itemMapper;
    
    @Override
    public Item save(Item item) {
        itemMapper.save(item);
        return item;
    }
    
    @Override
    public void update(Long itemId, ItemUpdateDto updateParam) {
        itemMapper.update(itemId, updateParam);
    }
    
    @Override
    public Optional<Item> findById(Long id) {
        return itemMapper.findById(id);
    }
    
    @Override
    public List<Item> findAll(ItemSearchCond cond) {
        return itemMapper.findAll(cond);
    }
}
```
* ItemRepository 인터페이스 구현
* ItemMapper에 작업 처리
* 스프링 빈으로 등록되어 서비스에서 사용 가능

<br>

### MyBatisConfig

```java
package hello.itemservice.config;

import hello.itemservice.repository.ItemRepository;
import hello.itemservice.repository.mybatis.ItemMapper;
import hello.itemservice.repository.mybatis.MyBatisItemRepository;
import hello.itemservice.service.ItemService;
import hello.itemservice.service.ItemServiceV1;
import lombok.RequiredArgsConstructor;
import org.springframework.context.annotation.Bean;
import org.springframework.context.annotation.Configuration;

@Configuration
@RequiredArgsConstructor
public class MyBatisConfig {
    private final ItemMapper itemMapper;
    
    @Bean
    public ItemService itemService() {
        return new ItemServiceV1(itemRepository());
    }
    
    @Bean
    public ItemRepository itemRepository() {
        return new MyBatisItemRepository(itemMapper);
    }
}
```

1. ItemMapper 자동 주입 (MyBatis 모듈이 생성한 프록시 객체)
2. MyBatisItemRepository 생성 및 등록
3. ItemService에 주입

<br>
